JAVASCRIPT KATA : Last !
====================

**Description:**

Find the last element of a list.

Example:

```
last( [1,2,3,4] ) // => 4
last( "xyz" )     // => z
last( 1,2,3,4 )   // => 4
```

In javascript and CoffeeScript a list will be an array, a string or the list of arguments.
